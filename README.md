## 项目结构说明

+ public：静态文件
+ src：源码
    - api 接口
	- components：公共组件
	- router：路由配置
	- store： 状态管理
    - http:   请求封装
	- views： 业务组件
		+ system 系统管理
		+ videoMonitor 监控管理
		+ errPage 错误界面
		+ log 日志
		+ login 登录
    - icon 图标库
